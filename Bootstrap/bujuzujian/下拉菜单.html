下拉菜单：下拉菜单是可切换的，是以列表格式显示链接的上下文菜单

这可以通过与 下拉菜单（Dropdown） JavaScript 插件 的互动来实现。
如需使用下列菜单，只需要在 class .dropdown 内加上下拉菜单即可

通过下拉菜单制作一个导航:父菜单类：dropdown-toggle，添加id,添加data-toggle="dropdown"

子菜单：class="dropdown-menu" role="menu" aria-labelledby="id"
