Bootstrap 面板（Panels）:面板组件用于把 DOM 组件插入到一个盒子中。创建一个基本的面板，只需要向 div元素添加 类 .panel 和 类 .panel-default 即可

默认的基本面板

面板的标题：通过以下两种方式来添加面板标题

使用 .panel-heading class 可以很简单地向面板添加标题容器
使用带有 .panel-title class 的 h1-h6 来添加预定义样式的标题。
panel-heading类添加标题
panel-body类添加内容
panel-title类添加标题
panel-body类添加内容

面板脚注：在面板中添加脚注，只需要把按钮或者副文本放在带有 class .panel-footer 的 div 中即可

panel-title类添加标题
panel-body类添加内容
panel-footer添加面板脚注
